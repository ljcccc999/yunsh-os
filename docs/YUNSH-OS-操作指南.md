# YUNSH OS v1.0.1 — 操作指南

> 系统所有操作的完整参考

## 一、鼠标操作

| 操作 | 功能 | 说明 |
|------|------|------|
| **左键点击** (图标) | 打开应用 | 主页/桌面图标点击启动 |
| **左键点击** (← 返回胶囊) | 返回上级 | Settings/About 等页面顶部 |
| **左键点击** (底部 Home 指示条) | 打开 App Switcher | 屏幕正下方的毛玻璃横条 |
| **左键上滑** (底部 Home 指示条) | 打开 App Switcher | 按住后向上拖动 >30px 触发 |
| **左键点击** (卡片) | 切换至该应用 | App Switcher 中的应用卡片 |
| **左键点击** (✕ 关闭按钮) | 关闭应用 | App Switcher 卡片左上角红色按钮 |
| **左键点击** (空白区域) | 返回桌面 | App Switcher 背景点击关闭 |
| **左键长按** (输出区域) | 弹出复制菜单 | Terminal 中选中文字后 |
| **左键长按** (输入框) | 弹出粘贴菜单 | Terminal / Browser 输入区域 |
| **右键点击** | 粘贴 | Terminal 输入栏 |

## 二、键盘快捷键

| 快捷键 | 功能 | 适用范围 |
|--------|------|----------|
| **Escape** | 返回上一级 / 关闭当前面板 | 全局 |
| **Print** | 截取全屏 | 全局 |
| **Ctrl + Shift + S** | 截取区域（选择区域截图） | 全局 |
| **Ctrl + Shift + C** | 打开/关闭 控制中心 | 全局 |
| **Ctrl + ↑** | 打开 App Switcher（任务切换器） | 全局 |
| **Ctrl + L** | 聚焦地址栏（Browser）/ 清屏（Terminal） | Browser / Terminal |
| **Ctrl + R** | 刷新页面 | Browser |
| **Ctrl + Shift + V** | 粘贴 | Terminal |
| **Ctrl + Shift + C**（选中文字时） | 复制 | Terminal |

## 三、窗口系统（MacWindow）

所有 App 在浮动圆角毛玻璃窗口中打开，多窗口可共存于纯黑背景。

| 操作 | 功能 |
|------|------|
| **拖拽标题栏** | 移动窗口位置 |
| 🔴 **红色按钮** | 关闭窗口 |
| 🟡 **黄色按钮** | 最小化窗口（App Switcher 中仍可选） |
| 🟢 **绿色按钮** | 全屏/恢复窗口 |
| **边缘拖拽**（左/右/下/角） | 缩放窗口大小 |
| **点击窗口** | 将该窗口置顶 |

### 📌 窗口钉选模式

标题栏右上角图钉按钮切换两种模式：

| 模式 | 图标 | 说明 |
|------|------|------|
| **悬停（Pinned）** | 白色图钉 | 窗口固定放置位置，屏幕滚动/转头时不动 |
| **视线跟随（Following）** | 青色发光图钉 | 窗口始终保持在视野中央附近 |

> ⚡ 未来：结合 IMU 3DoF 追踪后，悬停模式将根据头部旋转反向补偿，实现真正 AR 空间锚定

## 五、系统功能

### 截图
- **方式一**: 按 **Print** 键
- **方式二**: 点击状态栏截图图标
- **方式三**: 点击右下角悬浮截图按钮
- **方式四**: 按 **Ctrl+Shift+S** 区域截图
- **方式五**: 控制中心 📷 按钮
- 截图后右上角显示叠加预览

### 控制中心
- **方式一**: 按 **Ctrl+Shift+C**
- **方式二**: 点击状态栏左侧 ☰ 图标

### App Switcher (多任务切换)
- **方式一**: 点击 Home 指示条（屏幕底部毛玻璃横条）
- **方式二**: 从 Home 指示条**按住向上拖拽**
- **方式三**: 按 **Ctrl+↑**

### 待机
- 2 分钟无操作自动进入待机（Screensaver）
- 移动鼠标即可唤醒

### 激活向导（首次开机）
- 欢迎 → 选择语言 → 连接 Wi-Fi → 设置账户（可跳过）→ 初始化
- 跳过则使用默认账户 `yunsh:yunsh123`
